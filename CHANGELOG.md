# Changelog

## 0.1.2

- `주` 탭 일정 파서가 사용자별 DOM 차이에도 일정 블록을 더 안정적으로 찾도록 개선했습니다.
- 확장프로그램 실행 시 성공/실패 디버깅 로그를 `[FSD]` prefix로 출력하도록 했습니다.
- README 설치 옵션을 Git clone 우선, Release ZIP 보조 방식으로 정리했습니다.

## 0.1.1

- `주` 탭과 `내일정만보기` 기준으로 내 일정 집계를 지원하도록 개선했습니다.
